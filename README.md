## КОМАНДЫ

| Команда | Описание |
|---------|----------|
| `.выдать @user 100` | Выдать 100 валюты |
| `.bfgfarm 50` | Купить 50 видеокарт |
| `.bfgfarm 5м` | Автофарм каждые 5 минут |
| `.bfgfarm шахта` | Запуск/остановка шахты |
| `.bfgfarm стоп` | Остановить фарм |
